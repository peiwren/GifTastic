
### `GifTastic`
This is a fun gif search app. Once enter a fav animal name and submit, the app will populate with gifs of your choice.

### `Technology Used:`
HTML, CSS, JS

GIPHY API Search Endpoint: api.giphy.com/v1/gifs/search
